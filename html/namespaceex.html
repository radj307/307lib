Create an exception with a given message, using wide-characters.
This function only accepts types using wchar_t instead of char, however all characters are
converted to narrow chars before returning, as exception messages cannot contain wide chars.

This is an overloaded member function, provided for convenience. It differs from the above function only in what argument(s) it accepts.
