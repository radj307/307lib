template<std::derived_from< except > TReturn, typename... Ts>
requires std::constructible_from<TReturn, Ts...> WINCONSTEXPR TReturn ex::make_custom_exception_explicit ( Ts &&...  args)

Create a custom exception type from the given arguments.

This overload differs from make_custom_exception in that it passes the arguments directly to the TReturn type's constructor.
Because of this, it is required that TReturn is constructible from the types specified in the order that they are specified.

Template Parameters

...TsVariadic Templated Types.
Parameters
...argsArguments to pass to the constructor of TReturn.

If no such constructor exists, compilation fails.
