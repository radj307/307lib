) const
inline

Resolve a given filesystem path by checking, in order;
1. the current working directory,
2. the program location (if given in the constructor),
3. the PATH environment variable.
Calling this function is the equivalent of splitting the result of resolve() by its filename.
If no filename is included as part of the target path, the second return value is empty.

Parameters
pathThe target path to resolve. If the path can't be found, this is returned unmodified.
extensionsOptional list of filename extensions, if extensions are given, resolution will check if the given path exists with any of the given extensions.
Returns
std::pair<std::filesystem::path, std::filesystem::path>
