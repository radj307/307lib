enum token::DefaultDefs::LEXEME : char
strong

Represents the partially-abstracted type of a character. This acts as an intermediary between the raw character types and a Token Type, and LEXEMEs are not recognized by anything downstream from the TokenizerBase class & derivatives, including the Token struct.
A lexeme is the smallest unit in lexical analysis, and represents a character or sequence of characters.

◆ TokenType

enum token::DefaultDefs::TokenType : unsigned short
strong

This enum provides generic types that apply to some/all file formats. If a file format does not use one of the types listed here, it is safe to ignore it as types should never be required by any function.

Enumerator
NULL_TYPE 

Null type, indicates that a character is invalid or an error occurred.

UNKNOWN 

Unknown type, this should be the default return value of functions derived from TokenizerBase::getNext()

ESCAPED 

Escaped character, may appear inside of other tokens.

NEWLINE 

Newline / line feed.

END 

End of file.

DIVIDER 

Divider used by some languages (JSON ',') in lists to separate elements.

COMMENT 

Comment type, indicates a comment that can be ignored or used should that be needed.

STRING 

String type.

KEY 

Key type, used by some file formats (INI) that accept string values without enclosing quotes.

BOOLEAN 

Boolean type, should only ever match "true" or "false" (implementation-defined-case-sensitivity)

NUMBER 

Number type.

NUMBER_INT 

Integer type, used by some file formats (INI) to differentiate between floating-points and integrals.

HEADER 

Header type, used by some file formats (INI "[]") similarly to a key, but with special syntax.

SETTER 

Setter type, used by some file formats (INI '=', JSON ':') to indicate a variable being defined.

OPEN_SECTION 

Section open type, used by JSON & similar formats. Must have a matching CLOSE_SECTION.

CLOSE_SECTION 

Section close type, used by JSON & similar formats. Must have a matching OPEN_SECTION.

OPEN_ARRAY 

Array open type, used by JSON & similar formats. Must have a matching CLOSE_ARRAY.

CLOSE_ARRAY 

Array close type, used by JSON & similar formats. Must have a matching OPEN_ARRAY.
