Detailed Description

template<typename KeyType>
struct color::palette< KeyType >

Contains a key-color map and functions to set terminal colors inline.

Template Parameters
KeyType- The type of key to use as an identifier for each color. An enum of type char is recommended as each element only uses 1 byte.
