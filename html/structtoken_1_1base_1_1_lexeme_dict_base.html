Detailed Description

template<typename LexemeType>
struct token::base::LexemeDictBase< LexemeType >

Used to convert characters to a templated Lexeme object, which represents a single character's lexical "type".
A Lexeme can be thought of as a character's type, and as such multiple characters can be associated with a single Lexeme, but a single Lexeme always represents a single character length.
