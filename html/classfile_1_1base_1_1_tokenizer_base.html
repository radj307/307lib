Detailed Description

template<typename LexemeT, std::derived_from< LexemeDictBase< LexemeT >> Dictionary, typename TknType, std::derived_from< TokenBase< TknType >> TokenT>
class file::base::TokenizerBase< LexemeT, Dictionary, TknType, TokenT >

Base TokenizerBase object that accepts a stringstream and lexeme dictionary, then tokenizes the data.
Can be used inline by calling tokenize().

Template Parameters
LexemeTThe Lexeme Type to use. This is defined by a "definitions package".
DictionaryThe Lexeme Dictionary Type to use. This is defined by a "definitions package".
TknTypeThe TokenBase::Type to use. This is defined by a "definitions package".
TokenTThe TokenBase type to use. This is defined by a "definitions package".
