Detailed Description

template<var::streamable T, std::uint16_t INDENT_LENGTH = 2>
class term::legacy::print_tree< T, INDENT_LENGTH >

An ostream functor that prints the specified object as a nicely-formatted tree using DEC line drawing characters, without using recursion. See the constructor for more information.

Template Parameters
T- The type of object to print. It must have an operator<< for printing to a std::ostream.
INDENT_LENGTH- The length of the indentation for each subnode, in characters.
