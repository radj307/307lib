Detailed Description

template<typename T>
class color::format::FlagBase< T >

Base Formatting "flag" object, uses bit fields to store various types of formatting data.

Template Parameters
TVariable type to use. This essentially defines the maximum number of individual flags. Default is char (1 byte, max value 256)
