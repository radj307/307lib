Detailed Description

template<std::integral KeyType, var::any_same_or_convertible< ANSI::Sequence, ANSI::wSequence > SequenceType>
class term::basic_colorpalette< KeyType, SequenceType >

A simple color palette object that allows some useful features like a global color toggle and global color synchronization.
Custom functionality can be added by deriving from this object as most methods are marked virtual.

This is an alternative to the term::palette object that is based on the term::basic_colors object instead.
Use operator[] to access the colormap. Use operator() to access the default basic_colors operators.

Attention
This object's destructor automatically emits a reset escape sequence to the appropriate iostream for STDOUT. To disable this feature, set active to false before the object goes out-of-scope.
Template Parameters
KeyTypeAny integral indexer type to use as the palette key type.
SequenceTypeDetermines the character width used by escape sequence strings.
