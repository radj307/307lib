class var::basic_variant_map< TMap, TKey, Ts >


Template Parameters
TMapClass template that determines which associative container type to use.
Recommended types are std::map, std::unordered_map, std::multimap, etc.
TKeyA single, comparable type to use for the map keys.
Ts...Any number of independent types that the value of a key may represent.
