Detailed Description

template<typename T, std::derived_from< std::optional< T >> Base = casting_optional<T>>
struct opt::unsafe_optional< T, Base >

An "unsafe" wrapper object that allows implicit "casting"/retrieval of the underlying value.
If the value isn't set, an exception will be thrown during an implicit casting operation, making debugging very difficult.
For this reason, it is NOT recommended to use the unsafe_optional object in production code.

Template Parameters
TValue Type
BaseBase optional type. This defaults to casting_optional, but any type derived from std::optional is accepted.
