Detailed Description

Contains a wide variety of cross-platform ANSI escape sequences for every occasion.

Author
radj307


Covers most of the 'virtual sequences' documented by microsoft here:
https://docs.microsoft.com/en-us/windows/console/console-virtual-terminal-sequences
All of the sequences listed at the above link are usable on both Windows & UNIX-based operating systems.

std::string term::query::getResponse ( const char  seqEnd,
const size_t  bufSize = 8 
)
inlinenoexcept

Receive a query response from STDIN, and return it as a string. Any prior data waiting in STDIN is discarded.

Parameters
seqEnd- The final character in the expected ANSI sequence.
bufSize- The minimum number of bytes to reserve in the buffer. If this is insufficient, the buffer will be expanded with a potential reallocation penalty. The default is 8.
Returns
The query response as a std::string
