Stream insertion operator for the xLogs class.
Allows using xLogs in the same way as a std::ostream, with some important changes:
- A message level must be specified before messages can be inserted, unless prefixes are disabled.
- Messages are terminated with endm / nullptr.
- Messages are only sent to the output target once endm has been received.
