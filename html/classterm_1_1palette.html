Detailed Description

template<typename Key>
class term::palette< Key >

Associative binding container with additional helper methods & operators.
Maps color sequences to a templated key type.
Provides convenience methods for recalling color sequences by using keys.
Very useful when you want to control the usage of escape sequences throughout your program.
