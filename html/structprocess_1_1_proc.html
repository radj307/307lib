Detailed Description

template<size_t BUFFER_SIZE = 2048>
struct process::Proc< BUFFER_SIZE >

Execute a process, capture STDOUT & STDERR, and retrieve the return code.
Note that to retrieve the return code, you must call close() once the process has completed.

Template Parameters
BUFFER_SIZEThe size of the character array buffer used to read/write from the process pipe. Defaults to 2048.
