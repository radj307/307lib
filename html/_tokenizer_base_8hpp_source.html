1 /**
2  * @file TokenizerBase.hpp
3  * @author radj307
4  * @brief Contains the TokenizerBase class; a generic virtual tokenizer.
5  *\n ## Description #
6  *\n The tokenization process is split into 2 distinct phases:
7  *\n - __Scanner Stage__ _char -> LEXEME_
8  *\n - __Actual Tokenization__ _LEXEME -> TokenType_
9  *\n The TokenizerBase class should be specialized by inheritence.
10  *\n Derived objects should override the virtual __getNext__ function that handles the __Actual Tokenization__ _LEXEME -> TokenType_ phase.
11  *\n Other functions may also be overridden, but most file formats shouldn't require it.
12  *\n See the TokenizerINI class for an example.
13  *\n
14  *\n ## Standards & Requirements #
15  *\n - All token containers must have a token with _TokenType::END_ at the end, to represent EOF.
16  *\n _This is handled by the __tokenize__ function, which appends a blank token with TokenType::END._
17  *\n - The virtual __getNext__ function must be self-contained, and not require the end user to call other functions.
18  *\n _Excluding the __hasMore__ function,_
19  *\n - The virtual __getNext__ function may use recursion.
20  *\n - The virtual __getNext__ function must return a fully-constructed __Token__ object, and contain the string used to detect it.
21  *\n - It is recommended that the tokenization process does not verify whether a file is correctly formatted, only that tokens are correctly formatted: For example, quotation marks & brackets are closed properly. The format verification process should occur during the parsing process.
22  *\n - Derived objects _should_ be members of the token namespace, or a sub-namespace.
23  */

241  [[nodiscard]] std::vector<TokenT> tokenize_internal(const size_t& reserve)
242  {
243  std::vector<TokenT> tokens;
244  tokens.reserve(reserve);
245  while (ss)
246  tokens.emplace_back(std::move(getNext()));
247  if (!tokens.back().is_eof()) // append an EOF token if there isn't one already.
248  tokens.emplace_back(TokenT{ TokenType::END });
249  tokens.shrink_to_fit();
