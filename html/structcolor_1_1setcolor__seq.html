Detailed Description

template<typename SeqT = ANSI::Sequence>
struct color::setcolor_seq< SeqT >

Acts as a wrapper and controller for SGR & RGB color codes, as well as certain SGR format codes.
Uses the ostream operator<< to insert escape sequences into streams.

Template Parameters
SeqTThe type of string to use for escape sequences.
