Detailed Description

Logging object that automatically redirects std::clog to itself to allow capturing the standard log stream.
This is a buffered object, and does not have any output sources by default; To retrieve data from the log,
use the (<<|>>) operators or assign an automatic output source with the setAutoOut() function.
