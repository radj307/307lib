Detailed Description

Contains the ColorPalette struct, which provides a convenient API for applying consistent colors throughout a program.

Author
radj307

Example Implementation:

enum class MyKeyType { TYPE1, TYPE2, TYPE3, }

using Palette = color::ColorPalette<MyKeyType>;

Palette myPalette{ { { MyKeyType::TYPE1, color::red }, { MyKeyType::TYPE2, { color::green, color::Layer::FOREGROUND, color::FormatFlags::BOLD | color::FormatFlags::UNDERLINE } }, { MyKeyType::TYPE3, color::blue }, } };

void example() { std::cout << myPalette(MyKeyType::TYPE1) << "red text" << color::reset; }
