1413  *\n __Argument Types__
1414  *\n - Parameters are any arguments that do not begin with a dash '-' character that were not captured by another argument type.
1415  *\n - Options are arguments that begin with 2 dash '-' characters, and can capture additional arguments if the option name appears in the capture list.
1416  *\n - Flags are arguments that begin with a single dash '-' character, are a single character in length, and can capture additional arguments. Flags can appear alone, or in "chains" where each character is treated as an individual flag. In a flag chain, only the last flag can capture additional arguments.
1417  *\n __Capture Rules__
1418  *\n - Only options/flags specified in the capture list are allowed to capture additional arguments. Capture list entries should not include a delimiter prefix.
1419  *\n - Options/Flags cannot be captured under any circumstance. ex: "--opt --opt captured" results in "--opt", & "--opt" + "captured".
1420  *\n - If a flag in a chain should capture an argument (either with an '=' delimiter or by context), it must appear at the end of the chain.
1421  *\n - Any captured arguments do not appear in the argument list by themselves, and must be accessed through the argument that captured them.
1422  * @param args Commandline arguments as a vector of strings, in order and including argv[0].
1423  * @param captures A CaptureList instance specifying which arguments are allowed to capture other arguments as their parameters
1424  * @returns ArgContainer
1425  */
1426  inline arg_container parse(std::vector<std::string>&& args, const CaptureList& captures)
1427  {
1428  // remove empty arguments, which are possible when passing arguments from automated testing applications
1429  args.erase(std::remove_if(args.begin(), args.end(), [](auto&& s) { return s.empty(); }), args.end());
1430 
1431  arg_container cont{};
1432  cont.reserve(args.size());
1433 
1434  for (std::vector<std::string>::const_iterator it{ args.begin() }; it != args.end(); ++it) {
1435  auto [arg, d_count] { captures.parsingRules.stripPrefix(*it, 2ull) };
1436
